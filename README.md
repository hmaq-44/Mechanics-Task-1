# Mechanics-Task-1
Quadruped Robot Dog – Preliminary Mechanical Design

Overview

This repository contains the preliminary mechanical design deliverable for a simple quadruped (four-legged) robot dog. The goal of this task is not to build an advanced robot, but to demonstrate a solid understanding of the core mechanical fundamentals that make a legged robot able to stand and walk.

Task Description

The student is required to produce the initial mechanical design of a simple robot dog, with focus on the following points:


Body and frame shape – overall chassis geometry and structure.
Leg design – link configuration and dimensions.
Number of joints and degrees of freedom (DOF) – per leg and in total.
Motor/actuator selection – type and justification.
Preliminary torque calculation for at least one joint.
Stability and center of gravity (COG) analysis.
Proposed walking method (gait).
Expected mechanical problems/challenges.


Deliverable

A 2-page PDF report (robot_dog_design.pdf) containing:


Page 1: A 3D concept sketch (isometric illustration) of the robot dog showing the rectangular chassis and the four legs, with the hip, knee, and foot joints clearly labeled.
Page 2: A detailed write-up covering all 8 required points above, including a worked torque calculation and a joint/DOF summary table.


Task Details (as given)


Title: Robot Dog Design
Duration: 6 days (due before midnight Friday)
Submission: Upload link and deadline are posted on the course platform
Execution method: Open to any design tool (e.g., SketchUp) or specialized CAD software
Target audience: Mechanical-track students, or those aiming to pursue a "Full Stack" certificate


Notes

The objective is not to build an advanced, production-ready robot — it is to demonstrate understanding of the basic mechanical principles that enable a legged robot to stand and walk.
